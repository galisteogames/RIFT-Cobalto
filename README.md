# RIFT-Cobalto

Novo tema ALEKFULL para EmuELEC / Batocera / Retrobat / Batocera Plus
![Imagem do WhatsApp de 2025-12-06 à(s) 20 43 47_e215379d](https://github.com/user-attachments/assets/3a4e1c62-8285-43bd-b884-b812b74cb41e)

Criado por Fagner (AlekFull), exclusivo para as imagens Galisteo Cobalto. RIFT-Cobalto promete uma nova experiência em temas para retrogames. 

# Opções de visualização da lista de jogos

RIFT-Cobalto vem com 10 diferentes opções de visualização da lista de jogos (Coinops padrão, Coinops Detalhado, NFLIX, Gamecarousel e etc.).
