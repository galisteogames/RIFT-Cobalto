# RIFT-Cobalto

Novo tema ALEKFULL para EmuELEC / Batocera

Criado por Fagner (AlekFull), exclusivo para as imagens Galisteo Cobalto. RIFT-Cobalto promete uma nova experiência em temas para retrogames. 

# Opções de visualização da lista de jogos

RIFT-Cobalto vem com 10 diferentes opções de visualização da lista de jogos (Coinops padrão, Coinops Detalhado, NFLIX, Gamecarousel e etc.). 


# Artes exclusivas e suporte a mais de 200 sistemas

Todas as artes foram feitas respeitando os jogos icônicos da plataforma representada de forma única.

# Artes em movimento

Todos os sistemas apresentam um personagem clássico em destaque e em movimento.

# Muitas outras opções

RIFT-Cobalto tem 4 opções de idiomas diferentes, que são modificados automaticamente, pelo idioma que estiver no sistema.
Tela de informações dos jogos com muito mais detalhes
